# VM Cloud Infrastructure Lab

## Overview

This project is a hands-on simulation of a production-style cloud infrastructure built entirely using Oracle VirtualBox.

Rather than relying on a public cloud provider, the goal is to understand the underlying infrastructure by designing, deploying, securing, and managing multiple Linux virtual machines that work together as a highly available environment.

The project follows real-world infrastructure engineering practices, with continuous documentation, version control, and testing throughout the build process.

---

## Objectives

- Build a production-style virtual infrastructure
- Configure secure networking between virtual machines
- Deploy redundant web servers
- Configure a load balancer
- Deploy a MySQL database server
- Implement server hardening and security best practices
- Configure monitoring and logging
- Automate administrative tasks using Bash
- Document every stage of the project
